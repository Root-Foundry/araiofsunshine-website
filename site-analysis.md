# A Rai of Sunshine - Website Redesign Analysis

## Current Site Overview
**URL:** https://araiofsunshineofnorthfloridainc.org

**Organization:** A Rai of Sunshine of North Florida Inc.  
**Purpose:** Enhances the lives of individuals with disabilities in Jefferson County and surrounding areas by providing essential support and resources.

---

## Current Issues Identified

### 1. **Navigation Complexity**
- Too many navigation links (14+ main links)
- Duplicate/inconsistent formatting (uppercase vs. sentence case)
- No clear hierarchy or grouping

### 2. **Content Organization**
- Information is scattered across multiple pages
- Key services not prominently displayed
- Mission statement could be more prominent

### 3. **Readability Concerns**
- Long link names that could be simplified
- Newsletter signup could be more integrated
- Testimonial placement needs better visual hierarchy

---

### **C. Design Improvements**

#### Typography
- **Headers:** Large, bold, easy to read (Arial, Helvetica, or similar sans-serif)
- **Body Text:** 16-18px for better readability
- **Line Height:** 1.6-1.8 for comfortable reading
- **Color Contrast:** Ensure WCAG AA compliance (especially important for accessibility-focused org)

#### Color Scheme Recommendations
- **Primary:** Sunshine Yellow/Gold (#FFD700) - represents hope and optimism
- **Secondary:** Calm Blue (#4A90E2) - trust and stability
- **Accent:** Warm Orange (#FF8C42) - energy and support
- **Text:** Dark Gray (#333333) on white background
- **Backgrounds:** White with subtle warm cream accents (#FFF9E6)

#### Visual Elements
- Use icons for the three core services
- Add hero image showing community support
- Include photos from events (from the gallery)
- Ensure all images have alt text for accessibility

---

### **D. Content Strategy**

#### Homepage Content Hierarchy:
1. **Clear Value Proposition** (What you do, who you help)
2. **Three Core Services** (Easy to understand cards)
3. **How to Get Involved** (Clear CTAs)
4. **Social Proof** (Testimonials, sponsors)
5. **Newsletter Signup** (Stay connected)

#### Page-Specific Improvements:

**Services Page:**
- Create visual cards for each service
- Add "Learn More" buttons
- Include contact information for each service

**About Page:**
- Tell the story compellingly
- Include founder's story (if relevant)
- Team photos with bios

**Donate Page:**
- Explain impact of donations
- Show transparency (how funds are used)
- Multiple giving options clearly displayed
- Monthly vs. one-time options

**Resources Page:**
- Organize by category
- Add search functionality if many resources
- Include downloadable PDFs/guides

---

### **E. Mobile Responsiveness**
- Hamburger menu for mobile
- Touch-friendly buttons (44px minimum)
- Simplified layout for small screens
- Easy access to "Call" and "Donate" buttons

---

### **F. Accessibility Features** (Critical for this organization!)
- WCAG 2.1 AA compliance minimum
- Screen reader friendly
- Keyboard navigation
- Alt text for all images
- Closed captions for videos
- High contrast mode option
- Font size adjustment option

---

### **G. Performance Optimization**
- Compress images
- Minimize plugins
- Use caching
- Fast loading times (< 3 seconds)

---

## Recommended WordPress Theme Options

1. **Astra Pro** - Highly customizable, fast, accessible
2. **OceanWP** - Non-profit friendly, good documentation
3. **Kadence** - Modern blocks, great for service organizations
4. **GeneratePress** - Lightweight, excellent performance
5. **Neve** - Clean, AMP-ready, good for non-profits
